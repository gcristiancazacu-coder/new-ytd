# YT Downloader Pro

Aplikație mobilă profesională pentru descărcarea de videoclipuri/audio de pe YouTube și alte platforme.

## Arhitectură

- **Backend**: Python FastAPI (descărcări, procesare)
- **Frontend**: Flutter + Dart (interfață Android)

## Setup Rapid

### Backend
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### Flutter (după instalare Flutter SDK)
```bash
cd flutter_app
flutter pub get
flutter run
```
